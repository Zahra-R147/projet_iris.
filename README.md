# Classification des fleurs d'Iris
Ce projet a pour objectif de construire un modèle simple de machine learning permettant de classifier les fleurs d'Iris (Setosa, Versicolor, Virginica) en utilisant leurs caractéristiques morphologiques.

## Structure du projet
- donnees/ : contient les données (si ajoutées plus tard)
- notebooks/ : contient le notebook Jupyter Iris_Classification.ipynb
- scripts/ : contient les scripts Python (optionnel)
- resultats/ : contient les résultats exportés (CSV, figures)

## Outils utilisés
- Python 3
- NumPy
- Pandas
- Matplotlib
- Seaborn
- scikit-learn
- Jupyter Notebook
- Git & GitHub

## Étapes principales
1. Chargement du dataset Iris
2. Analyse exploratoire des données (EDA)
3. Préparation des données
4. Entraînement du modèle KNN
5. Évaluation du modèle
6. Sauvegarde des résultats
